# SharePoint File Upload Project

This project allows you to upload a file using a placeholder link and append its metadata to a SharePoint list via a Python script.

## Structure
- upload_script.py: Main script for uploading and appending to SharePoint
- README.md: Project instructions

## Requirements
- Python 3.x
- SharePoint access (with API permissions)
- Required Python packages: requests, office365-rest-python-client

## Usage
1. Replace the placeholder link with your actual file upload link.
2. Run the script to upload the file and append its metadata to the SharePoint list.

## Setup
Install dependencies:
```
pip install requests office365-rest-python-client
```
